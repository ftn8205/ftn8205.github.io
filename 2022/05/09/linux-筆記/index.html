第二章 系統操作篇

ls

  • -l 詳細列出所有文件的訊息
  • -r 照檔名倒序排列
  • -t 照時間順序排序
  • -a 顯示隱藏文件與一般文件
  • -R 把文件夾內的資料也都顯示出來

cd

  • cd /path/to/… 絕對路徑
  • cd ./path/to/… 相對路徑
  • cd ../path/to/.. 相對路徑

文件夾新增刪除

  • mkdir 新增新目錄
  • rmdir 刪除空目錄
  • rm -r 刪除非空目錄

cp, mv

  • cp 原文件位置 新文件位置 (複製原文件到其他位置
  • mv 原檔名 新檔名 (更改檔名
  • mv 原文件 新位置 (移動檔案到新位置

文本查看命令

  • cat 文本內容顯示到終端
  • head 查看文件開頭前10行
  • tail 查看文件結尾10行 (可用 -f 當文件更新時，同步顯示更新)
  • wc 統計文件內容訊息

打包壓縮文件

  • tar cf 打包後的文件 要打包的文件 [ tar cf ./abc.tar upppppp]
  • tar czf 打包後的文件 要打包的文件 (打包並壓縮)
  • c 打包
  • x 解包
  • f 指定操作類型為文件

Vim

  • g 到第一行
  • G 到最後一行
  • dd 刪除該行
  • u 復原
  • shift 6 到某行的開頭
  • shift 4 到某行的結尾
  • :wq 儲存並離開
  • :! 執行linux指令 (:! ls)
  • /字符名稱 查找某個字符，按n可以跳到下一個匹配的地方, shift n 跳到上一個匹配的地方
  • :set nu 顯示行號
  • :set nonu 拿掉行號

用戶與權限管理

  • useradd 新建用戶
  • userdel 刪除用戶
  • passwd 修改用戶密碼
  • usermod 修改用戶屬性
  • chage 修改用戶屬性
  • su - user1 臨時切換成user1的角色
  • sudo 以其他角色身份執行命令 (要先用root權限，在visudo中設定，允許讓某個角色做這什麼事情)

/etc/passwd

第一個字: 用戶名

第二個字: 是否需要密碼驗證 (x: 不用密碼)

第三個字: 用戶ID

第四個字: groupID

第五個字: 家目錄的位置

第六個字: 所使用的命令解釋器

/etc/shadow

第一個字: 用戶名

第二個字: 用戶加密後的密碼

/etc/group

第一個字 組名

第二個字: 組密碼

第三個字: 組ID

文件與目錄權限

b: 硬碟

c: 終端
